OFCTv2 스킨은 OFCT 방송과 제작자 yhsphd의 개인적 사용을 위해 만들어진 스킨이지만,
· 사운드 파일
· 그림
을 제외한 파일은 제가 제작하였고, 자유롭게 사용하실 수 있습니다.
자유로운 변형도 가능하며, 배포도 가능하나 배포 시 GitHub 리포지토리 주소
https://github.com/yhsphd/OFCTv2 를 표키해주세요!

GitHub에서 리포지토리 전체를 다운로드하면
스프라이트 생성에 사용된 psd(포토샵), aep(애프터 이펙트) 프로젝트 파일을 함께 받을 수 있습니다.
(psd파일 안에 참고 용도로 사용된 다른 스킨의 이미지가 들어있을 수도 있으나 모두 최종 이미지 출력에는 사용되지 않음)

hit300g(Geki, 激) 파일이 제작되어 있지만, 플레이에 방해되지 않도록 빠져 있습니다.
스킨 폴더 내 hit300g 폴더의 파일들을 스킨 폴더로 옮겨주시면 사용 가능합니다.

스킨의 문제나 개선 사항 등 의견은 카페/밴드 댓글, GitHub 이슈, 혹은 yhsphd#1952 디코로!
